# Question Gate

Use this reference when deciding whether to ask, research, assume, or act.

## Gate

Ask a question only when all of these are true:

1. **Decision delta**: a plausible answer changes direction, scope, cost, risk, authority, or success criteria.
2. **Unresolved evidence**: the answer is not reliably available in supplied material, approved project context, or safe research.
3. **User ownership**: the user owns the preference, fact, authority, or tradeoff.
4. **Timely consequence**: acting without the answer would cause material rework, a misleading result, or an unsafe external action.

## Choose the Alternative

| Condition | Default action |
| --- | --- |
| The evidence is available | Read or research it. Do not ask. |
| The detail does not change a decision | Use a reasonable convention and proceed. |
| The uncertainty is low-risk and reversible | State a minimal assumption and proceed. |
| The uncertainty creates an external, financial, sensitive, or irreversible effect | Ask or stage the action for approval. |
| A durable preference is relevant but context may have changed | Ask one revalidation question. |

## Write the Question

Name the decision, not the missing field. Prefer:

> To choose the investor list, should I optimize for seed-stage climate funds in North America, or a different profile?

Avoid:

> What is your stage, geography, sector, ticket size, traction, deck link, and timeline?

When a choice is common and consequential, give a short recommendation with alternatives. Never use the gate to demand information an agent can discover itself.
